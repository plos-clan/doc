* 最初的系统名叫 `CrashPowerDOS for x86` 后觉得太长改名
* XIAOYI12对OS的各种模块做了高度解耦, 所以将该系统的源码移植到您的项目非常容易
* CoolPotOS沿用了很多PlantOS的技术栈, 包括文件系统, 硬盘驱动接口, VFS
